1. command "tee" - принимает на вход строку с клавиатуры и выводит в стандартный поток эту же строку
2. command "echo" - принимает аргументы в виде строк через пробел и выводит их в стандартный поток вывода
3. "ctrl + d" - символ, говорящий о том, что дальше ввода не будет => программа завершаетсяс кодом 0
4. operator "|" - перенаправляет поток вывода одной комманды в поток ввода другой (pipe)
5. operator "<<" - перенаправляет результат правой программы в стандартный поток вывода левой
6. operator "<" - читать из файла
7. operator ">" - перезаписывать в файл(старые данные удаляются из фацла до начала работы всей программы)
8. operator ">>" - дозаписать в файл
9. чтобы писать один поток в другой, можно сделать так: <№потока>& >/>>

Ex-s:

echo lol | tee
result: lol

tee << (echo lol)
result: lol

у функции int main() может быть два аргумента: int arc, const char** argv
